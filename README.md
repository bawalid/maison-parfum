# Lux Watches

Luxury watch e-commerce web app built with a MERN structure:

- `backend`: Express + MongoDB API with seed script
- `frontend`: React + Vite + Tailwind CSS storefront
- `resources`: reusable design and content system for rebuilding the same experience

## Quick start

1. Install dependencies in `backend` and `frontend`.
2. Copy `backend/.env.example` to `backend/.env`.
3. Set your MongoDB connection string.
4. Run the seed script.
5. Start backend and frontend servers.

## Scripts

- `npm run dev:backend`
- `npm run dev:frontend`
- `npm run seed`
- `npm run build`
